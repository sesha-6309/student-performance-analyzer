# student-performance-analyzer
Python program to analyze student exam marks and classify them as Excellent, Very Good, Good, Average, or Fail. Validates marks (0–100) and counts total valid and failed students. Includes personalized logic: if the entered name length is even, scores between 35–39 are upgraded to 40 as grace marks.
 
 Features

Accepts number of students and their marks

Validates marks (must be between 0 and 100)

Classifies performance as:

Excellent (90–100)

Very Good (75–89)

Good (60–74)

Average (40–59)

Fail (0–34)

Displays total valid students and total failed students

 Personalized Logic

If the entered name has an even length, students scoring between 35 and 39 are automatically upgraded to 40 (grace marks).

 Technologies Used

Python

Loops

Conditional statements

Lists

 How to Run

Run the Python file.

Enter number of students.

Enter name for personalized logic.

Enter marks when prompted.

View performance classification and summary.
